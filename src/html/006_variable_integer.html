Universal Language Repository 006: Memory Storage (Integer Variables)

We have successfully stored text. Now, we must store a mathematical value. Let's create a variable to store the number of Photon Torpedoes our ship has in its armory, and then command the engine to print that number.

This reveals a major difference in how system languages handle memory. Because numbers are processed differently than words, low-level languages require us to explicitly declare the variable as an int (Integer) so the CPU knows exactly how to allocate the RAM.

1. JavaScript (HTML / Browser)

JavaScript remains flexible. We use the same let keyword we used for Strings. Because we typed a raw number without quotes, the engine automatically stores it as a mathematical value. 

<!DOCTYPE html>
<html>
<head>
    <script>
        let photonTorpedoes = 50;
        console.log(photonTorpedoes);
    </script>
</head>
<body>
</body>
</html>

2. JavaScript (Node.js)

The Node.js environment uses the exact same dynamic type inference. 

let photonTorpedoes = 50;
console.log(photonTorpedoes);

3. Python

Python requires no keywords. You assign the number directly to the variable name, and the engine handles the memory allocation under the hood. 

photonTorpedoes = 50
print(photonTorpedoes)

4. C

Here we see the strict hardware rules. We cannot just invent a variable name. We must explicitly write int before the variable to tell the C compiler to lock in a specific chunk of numerical memory. We then use our %d placeholder to retrieve it. 

#include <stdio.h>

int main() 
{
    // Explicitly declare the variable as an Integer
    int photonTorpedoes = 50;
    
    // Retrieve the variable using the decimal integer placeholder
    printf("%d\n", photonTorpedoes);
    return 0;
}

5. C++

C++ requires the exact same explicit memory declaration as C. We use the int keyword, but the std::cout stream makes retrieving and printing the variable much safer and easier than legacy C. 

#include <iostream>

int main() 
{
    // Explicitly declare the variable as an Integer
    int photonTorpedoes = 50;
    
    std::cout << photonTorpedoes << std::endl;
    return 0;
}

6. Rust

Rust shows off its modern design. It uses the let keyword and infers that 50 is an integer. Behind the scenes, the Rust compiler locks it into strict numerical memory for maximum safety, giving you the speed of C++ with the ease of JavaScript. 

fn main() 
{
    let photonTorpedoes = 50;
    
    println!("{}", photonTorpedoes);
}

Engineering Summary

Statically typed languages (C, C++) require you to use the int keyword to declare an integer variable. If you tried to put a word into an int variable, the C++ engine would refuse to compile and throw an error, preventing you from crashing the ship's systems in deep space!
