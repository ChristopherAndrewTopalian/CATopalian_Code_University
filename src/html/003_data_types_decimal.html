Universal Language Repository 003: Transmitting Decimals

We have successfully transmitted whole numbers, but space is rarely measured in perfect integers. Whether we are calculating shield harmonics or plotting a course at Warp 9.97, we need to transmit decimal numbers.

In computer science, a decimal number is called a Floating Point number (or a "Float"). If it requires extreme mathematical precision, it is called a Double. Let's see how our different systems handle decimal transmissions.

1. JavaScript (HTML / Browser)

JavaScript makes this incredibly easy. Under the hood, JavaScript treats all numbers as high-precision decimals automatically. 

<!DOCTYPE html>
<html>
<head>
    <script>
        console.log(9.97);
    </script>
</head>
<body>
</body>
</html>

2. JavaScript (Node.js)

The Node.js desktop engine treats the decimal the exact same way. No special formatting is required. 

console.log(9.97);

3. Python

Python also handles floating-point numbers seamlessly without requiring any additional syntax. 

print(9.97)

4. C

In the legacy C language, we must use a brand new placeholder. If we try to use %d (for integer), the engine will crash or print garbage data. We must explicitly use %f to tell the compiler: "Expect a floating-point decimal here." 

#include <stdio.h>

int main() 
{
    // %f is the placeholder for a floating-point decimal
    printf("%f\n", 9.97);
    return 0;
}

5. C++

Here we see the safety and ease of modern C++. Instead of forcing you to memorize the %f placeholder, the cout stream automatically detects that 9.97 is a decimal and handles the memory formatting for you safely. 

#include <iostream>

int main() 
{
    std::cout << 9.97 << std::endl;
    return 0;
}

6. Rust

Rust maintains its modern, safe architecture. You can use the exact same empty brackets {} that you used for integers. The Rust compiler analyzes the number and automatically formats the decimal for transmission. 

fn main() 
{
    // The {} placeholder automatically adapts to decimals
    println!("{}", 9.97);
}

Engineering Summary

Notice how C++, Rust, Python, and JavaScript all work to reduce your cognitive load by formatting the output automatically. Only C requires you to explicitly manage the difference between an integer (%d) and a float (%f) during a print command.
