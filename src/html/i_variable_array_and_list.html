Arrays and Lists

So far, we have only stored a single piece of data inside a variable. But what if Starfleet Command asks us to transmit a list of active starships? We must create a Data Collection, commonly known as an Array or a List.

This lesson reveals a massive difference between scripting languages and system languages. Scripting engines are smart enough to look at an entire list and print it to the screen all at once. Low-level system languages require you to manually loop through the list, printing one ship at a time.

JavaScript (HTML / Browser)

JavaScript uses square brackets [] to create an Array. The console.log() command is powerful enough to print the entire array structure to the screen at once. 

<!DOCTYPE html>
<html>
<head>
    <script>
        let fleet = ["Enterprise", "Defiant", "Voyager"];
        console.log(fleet);
    </script>
</head>
<body>
</body>
</html>

JavaScript (Node.js)

The desktop engine operates exactly the same, easily printing the entire collection. 

let fleet = ["Enterprise", "Defiant", "Voyager"];
console.log(fleet);

Python

Python calls these "Lists". It also uses square brackets and can print the entire list to the console natively. 

fleet = ["Enterprise", "Defiant", "Voyager"]
print(fleet)

Lua

Lua is highly optimized for game scripting. Instead of Arrays, Lua uses Tables, defined by curly brackets {}. Natively printing a table just shows its memory address, so we use a built-in command called table.concat to glue the ships together with commas for printing. 

fleet = {"Enterprise", "Defiant", "Voyager"}
-- Concatenate (glue) the table together with commas
print(table.concat(fleet, ", "))

C

C does not have a built-in list printer. We create an Array of Strings (using pointers *), and we must manually use a for loop to iterate through the memory and print each ship one by one. 

#include <stdio.h>

int main() 
{
    const char *fleet[] = {"Enterprise", "Defiant", "Voyager"};
    
    // Manually loop through the 3 memory slots
    for(int i = 0; i < 3; i++) 
    {
        printf("%s ", fleet[i]);
    }
    
    return 0;
}

C++

C++ provides a powerful, expandable list called a std::vector. Like C, we still must loop through it to print the contents, but C++ offers a highly readable modern loop syntax to do so. 

#include <iostream>
#include <vector>
#include <string>

int main() 
{
    std::vector<std::string> fleet = {"Enterprise", "Defiant", "Voyager"};
    
    // Loop through each ship in the fleet
    for(const auto& ship : fleet) 
    {
        std::cout << ship << " ";
    }
    
    return 0;
}

C#

C# is a heavy-duty language built for enterprise apps and 3D game engines. Like Lua, we can use a built-in Join command to quickly fuse the array together into a single readable string for the console. 

using System;

class Program 
{
    static void Main() 
    {
        string[] fleet = {"Enterprise", "Defiant", "Voyager"};
        
        // Join the array elements with a comma
        Console.WriteLine(string.Join(", ", fleet));
    }
}

Rust

Rust arrays have a fixed size. To print an entire array at once, Rust utilizes a special "Debug Format" placeholder: {:?}. This tells the compiler to automatically unpack the array for us! 

fn main() 
{
    let fleet = ["Enterprise", "Defiant", "Voyager"];
    
    // The {:?} placeholder triggers debug mode to print the whole array
    println!("{:?}", fleet);
}

Engineering Summary

This tutorial bridges the gap between raw data and logic loops. In high-level languages, the engineers who built the language wrote the loops for you behind the scenes. In system languages like C and C++, you are the engineer—you must write the iteration loops yourself to access the memory.
