Universal Language Repository: Logic Gates (The 'if' Statement)

We know how to store variables. Now we must teach the computer how to make a decision based on those variables. To do this, we use the most fundamental Logic Gate in computer science: the if statement.

An if statement checks a condition. If the condition is True, the computer executes the code inside the block. If the condition is False, the computer ignores the code entirely. Let's write a script that checks if our ship's shields have dropped to 0, and if so, sounds a Red Alert!

Note: Notice the use of double equals == or triple equals ===. A single equals sign = is used to STORE data. Double or triple equals are used to COMPARE data.

JavaScript (HTML / Browser)

JavaScript uses parenthesis () for the condition, and curly brackets {} to hold the code that should execute. We use the strict === operator for safe comparisons. 

<!DOCTYPE html>
<html>
<head>
    <script>
        let shields = 0;
        
        if (shields === 0) 
        {
            console.log("RED ALERT: Shields are down!");
        }
    </script>
</head>
<body>
</body>
</html>

JavaScript (Node.js)

The desktop engine uses the exact same logic. (We assume you are running this from an open terminal or a .bat file, so we do not need a pause command). 

let shields = 0;

if (shields === 0) 
{
    console.log("RED ALERT: Shields are down!");
}

Python

Python is unique. It drops the parenthesis and curly brackets entirely! Instead, it uses a colon : to open the gate, and relies strictly on indentation (spaces) to know which code belongs inside the if statement. 

shields = 0

if shields == 0:
    print("RED ALERT: Shields are down!")

# Hold the terminal window open
input("Press Enter to exit...")

Lua

Lua uses the words then to open the gate and end to close it. It is highly readable and perfect for game scripting. 

shields = 0

if shields == 0 then
    print("RED ALERT: Shields are down!")
end

-- Hold the terminal window open
print("Press Enter to exit...")
io.read()

C

The grandfather of systems languages establishes the standard format: parenthesis for the condition, curly brackets for the code block. 

#include <stdio.h>

int main() 
{
    int shields = 0;
    
    if (shields == 0) 
    {
        printf("RED ALERT: Shields are down!\n");
    }
    
    // Hold the terminal window open
    printf("Press Enter to exit...");
    getchar();
    return 0;
}

C++

C++ utilizes the exact same syntax as C for its logic gates. 

#include <iostream>

int main() 
{
    int shields = 0;
    
    if (shields == 0) 
    {
        std::cout << "RED ALERT: Shields are down!" << std::endl;
    }
    
    // Hold the terminal window open
    std::cout << "Press Enter to exit...";
    std::cin.get();
    return 0;
}

C#

C# matches the C/C++ format perfectly, making transitioning between these heavy-duty languages very smooth. 

using System;

class Program 
{
    static void Main() 
    {
        int shields = 0;
        
        if (shields == 0) 
        {
            Console.WriteLine("RED ALERT: Shields are down!");
        }
        
        // Hold the terminal window open
        Console.WriteLine("Press Enter to exit...");
        Console.ReadLine();
    }
}

Rust

Rust is a modern hybrid. It drops the parenthesis around the condition to keep the code clean, but keeps the curly brackets {} to ensure the code block is strictly managed. 

use std::io;

fn main() 
{
    let shields = 0;
    
    if shields == 0 
    {
        println!("RED ALERT: Shields are down!");
    }
    
    // Hold the terminal window open
    println!("Press Enter to exit...");
    let mut buffer = String::new();
    io::stdin().read_line(&mut buffer).unwrap();
}
