Universal Language Repository 008: Memory Storage (Boolean Variables)

We have stored text, whole numbers, and decimals. To complete our fundamental memory systems, we must store Logic. Let's create a variable to track whether the ship's deflector shields are online (True).

This is called a Boolean variable. Tracking these true/false states is how we build AI decision trees. Remember that beneath the language syntax, the computer processor stores a True state in memory as the binary number 1.

1. JavaScript (HTML / Browser)

JavaScript uses the let keyword and automatically recognizes true as a logical state, storing it appropriately. 

<!DOCTYPE html>
<html>
<head>
    <script>
        let shieldsUp = true;
        console.log(shieldsUp);
    </script>
</head>
<body>
</body>
</html>

2. JavaScript (Node.js)

The desktop engine uses the exact same dynamic logic assignment. 

let shieldsUp = true;
console.log(shieldsUp);

3. Python

Python manages the memory automatically, but requires you to strictly capitalize the word True when assigning it to the variable. 

shieldsUp = True
print(shieldsUp)

4. C

Legacy C requires the inclusion of the stdbool.h library just to use the word true. We must explicitly declare the variable using the bool keyword, and we retrieve it using the integer placeholder %d, which will print 1. 

#include <stdio.h>
#include <stdbool.h>

int main() 
{
    // Explicitly declare the memory as a Boolean
    bool shieldsUp = true;
    
    // Prints the raw binary value (1)
    printf("%d\n", shieldsUp);
    return 0;
}

5. C++

Modern C++ has the bool keyword built directly into the language. We declare the memory type, and std::cout handles the output stream (though it will still print the raw 1 by default). 

#include <iostream>

int main() 
{
    // Explicitly declare the memory as a Boolean
    bool shieldsUp = true;
    
    std::cout << shieldsUp << std::endl;
    return 0;
}

6. Rust

Rust infers the logic state automatically using let, locking the variable into safe boolean memory and automatically translating the binary back to the English word "true" upon output. 

fn main() 
{
    let shieldsUp = true;
    
    println!("{}", shieldsUp);
}
