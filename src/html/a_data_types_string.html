Universal Language Repository: The Comm-Link

Welcome to the Universal Translator. A senior Starfleet engineer does not just know one language; they understand the underlying logic of all systems. Whether you are scripting a quick UI in the browser, writing server logic, or programming a low-level robotics interface, the logic remains the same, only the syntax changes.

In our first module, we will establish a basic Comm-Link. We are going to instruct six different system environments to print the exact same message: "Hi everyone".

JavaScript (HTML / Browser)

The standard for visual interfaces. We place our logic directly inside the <head> of the document using a script tag. To see this message, you open the HTML file in a browser and press F12 to view the console. 

<!DOCTYPE html>
<html>
<head>
    <script>
        console.log("Hi everyone");
    </script>
</head>
<body>
</body>
</html>

JavaScript (Node.js)

The exact same V8 engine as the browser, but stripped of the visual HTML wrapper. This is used for pure desktop data processing and massive server operations. Run via the system terminal. 

// Run in terminal with: node mission.js
console.log("Hi everyone");

Python

Known for its highly readable, minimalist syntax. Python is heavily utilized in deep space data science, mathematics, and machine learning models. 

# Run in terminal with: python mission.py
print("Hi everyone")

C

The grandfather of modern computing. C is a low-level systems language used to write operating system kernels and hardware drivers. It requires the Standard Input/Output library (stdio.h) to use the printf (print formatted) command. 

// Compile and run via terminal
#include <stdio.h>

int main() 
{
    printf("Hi everyone\n");
    return 0;
}

C++

The heavy-duty evolution of C. When you need absolute control over the ship's memory combined with complex object structures for robotics or graphical user interfaces, C++ is the industry standard. It utilizes the Input/Output Stream library (iostream). 

// Compile and run via terminal
#include <iostream>

int main() 
{
    std::cout << "Hi everyone" << std::endl;
    return 0;
}

Rust

A modern, ultra-secure systems language designed to prevent memory leaks and structural crashes. It uses an exclamation mark ! to denote that it is calling a system macro rather than a standard function. 

// Compile and run via terminal
fn main() 
{
    println!("Hi everyone");
}

Engineering Summary

Notice the structural differences. Scripting languages (JavaScript, Python) can execute commands immediately on the first line. System languages (C, C++, Rust) require a main wrapper function (int main() or fn main()) to act as the primary ignition switch for the compiled software.
