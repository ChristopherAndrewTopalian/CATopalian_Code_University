Universal Language Repository 004: Transmitting Logic (Booleans)

We have transmitted text (Strings) and math (Integers and Decimals). Now, we must transmit Logic.

To run a ship's diagnostic routine or build an AI, the engine must evaluate whether a statement is True or False. In computer science, this is called a Boolean data type (named after mathematician George Boole).

Underneath the hood, computer processors do not understand the words "true" or "false". They only understand binary: 1 (On) and 0 (Off). Watch closely how these different languages format a Boolean transmission.

1. JavaScript (HTML / Browser)

JavaScript prints the word exactly as it is typed, in all lowercase letters. 

<!DOCTYPE html>
<html>
<head>
    <script>
        console.log(true);
    </script>
</head>
<body>
</body>
</html>

2. JavaScript (Node.js)

The Node.js desktop engine behaves identically, printing the lowercase word. 

console.log(true);

3. Python

Warning: Python is unique. It requires Booleans to be capitalized. If you type true with a lowercase 't', the Python engine will crash. 

print(True)

4. C

The legacy C language reveals the hardware's truth. C does not have a built-in "boolean" word by default. We must import a special library (stdbool.h) just to use the word true. Even then, when we print it, we must use the integer placeholder %d, and the screen will output the number 1. 

#include <stdio.h>
#include <stdbool.h> // Required to use the word 'true'

int main() 
{
    // Prints the number 1 (Binary for True)
    printf("%d\n", true);
    return 0;
}

5. C++

Modern C++ has the word true built-in, but the cout stream still respects the raw hardware. By default, transmitting true will print the number 1 to the console. (You can force it to print the English word, but it requires advanced formatting). 

#include <iostream>

int main() 
{
    // Prints the number 1
    std::cout << true << std::endl;
    return 0;
}

6. Rust

Rust acts as a modern bridge. It understands the lowercase word like JavaScript, automatically translating the binary back into human-readable text on the console. 

fn main() 
{
    println!("{}", true);
}

Engineering Summary

This lesson exposes the "Metal" of the computer. High-level languages (JS, Python, Rust) hide the binary from you, showing you the English words "true" or "True". Low-level systems languages (C, C++) strip away the English and show you the raw electrical state of the memory: 1.
